Detailed Description

template<typename LTMCImpl>
class knowledge_rep::LTMCLock< LTMCImpl >

Represents a lock on the backing database.

Currently, there is no granularity in locking. Holding the lock will prevent all other knowledge_rep clients from performing any operations on the database.
